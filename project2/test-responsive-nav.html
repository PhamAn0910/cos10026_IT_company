Responsive Navigation Test

This page tests the responsive navigation implementation. Resize your browser window to see the changes:

  • At larger screen sizes: Navigation displays horizontally
  • At smaller screen sizes (less than 920px):
    • A hamburger menu (☰) appears
    • The navigation links are hidden by default
    • Clicking the hamburger toggles the navigation visibility

Implementation Details:

The responsive navigation includes:

  1. A hamburger button for toggling the menu
  2. CSS for responsive styling
  3. JavaScript for handling the toggle functionality
  4. Keyboard accessibility (press ESC to close)
  5. Overlay background when menu is open
